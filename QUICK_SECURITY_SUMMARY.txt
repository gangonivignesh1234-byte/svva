📥 BACKUP YOUR DATA
════════════════════════════════════════════════════════════════════════════

Browser Console Method (all browsers):
  1. Press F12 (Windows/Linux) or Cmd+Option+I (Mac)
  2. Go to Console tab
  3. Paste this code:

     const data = {};
     Object.keys(localStorage)
       .filter(k => k.startsWith('svva_'))
       .forEach(k => { data[k] = localStorage.getItem(k); });
     copy(JSON.stringify(data));

  4. Paste into Notes, email, or save as .txt file
  5. Keep backup safe on your computer

Settings Export:
  1. Open app → More (⚙️) → Settings
  2. Tap "📥 Export Data"
  3. Copy to clipboard
  4. Save to file

════════════════════════════════════════════════════════════════════════════
🗑️  DELETE YOUR DATA
════════════════════════════════════════════════════════════════════════════

Complete Deletion (Console):
  1. Press F12
  2. Go to Console
  3. Paste this:
     Object.keys(localStorage)
       .filter(k => k.startsWith('svva_'))
       .forEach(k => localStorage.removeItem(k));
  4. Refresh page

Or use Settings → Clear All Data

════════════════════════════════════════════════════════════════════════════
🔄 RESTORE FROM BACKUP
════════════════════════════════════════════════════════════════════════════

Console Method:
  1. Copy your backup JSON
  2. Press F12 → Console
  3. Paste backup JSON as: const backup = {your...data...};
  4. Then paste:
     Object.entries(backup).forEach(([k, v]) => {
       localStorage.setItem(k, v);
     });
  5. Refresh page

Or use Settings → Import Data
