## 📱 How to Open

### On **Mobile (Android or iPhone)**

1. Open **Chrome** (Android) or **Safari** (iPhone)
2. Visit: `https://gangonivignesh1234-byte.github.io/svva/`
3. **Install as app:**
   - **Android Chrome:** Menu (⋮) → "Add to Home Screen" → Install
   - **iPhone Safari:** Share (⬆️) → "Add to Home Screen" → Add

### On **Desktop (Mac/Windows/Linux)**

1. Open **Chrome**, **Edge**, or **Safari**
2. Visit: `https://gangonivignesh1234-byte.github.io/svva/`
3. **Install as desktop app:**
   - Look for the install icon (🖥️⬇️) in the address bar
   - Or: Menu → "Install SVVA"

## ✨ Features

- **10-step onboarding** with Prakriti (Ayurvedic constitution) detection
- **Home screen** with daily stats: Steps, Calories, Water, Fasting, Soma Engine score
- **5 Vedic protocols** with phases, daily timelines, and custom recommendations  
- **Gym tracker** with Mon-Sat split, exercise logging, session timer
- **Food log** with 35 Indian foods, mineral tracking
- **Profile & Wearables** for Apple Health, Google Fit, Bluetooth devices
- **Dark + Light mode** with smooth transitions
- **100% offline** after first load — all data stored locally on your device
- **PWA installable** — works like a native app
